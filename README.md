# Memodesk - Your Thoughts. Organized.

A clean and intuitive study notes organizer to help you manage your academic notes efficiently.

## Features

- **Add Notes** - Quickly create new study notes
- **Delete Notes** - Remove notes you no longer need
- **Search Notes** - Find your notes instantly with search functionality
- **Theme Switching** - Toggle between light and dark modes for comfortable studying at any time of day

## Tech Stack

- **React** - Built with React for a responsive and dynamic user interface
- **Web Application** - Access your notes from any browser

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/memodesk.git
```

2. Navigate to the project directory
```bash
cd memodesk
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm start
```

5. Open your browser and visit `http://localhost:3000`

## Usage

- **Creating a Note**: Click the "Add Note" button and enter your study content
- **Searching Notes**: Use the search bar to filter through your notes
- **Deleting Notes**: Click the delete button on any note to remove it
- **Changing Theme**: Toggle between light and dark mode using the theme switcher
